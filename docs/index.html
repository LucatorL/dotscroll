DotScroll

A tiny, zero-dependency scroll progress navigation. Drop it into any website and get beautiful dot indicators that track where you are on the page.

★ GitHub Quick Start →
02

Installation

Two files. That's it.

No npm required. Just link the CSS and JS files, and add the data-dotscroll attribute to your sections.

<!-- 1. Include CSS -->
<link rel="stylesheet" href="dotscroll.css">

<!-- 2. Add attributes to your sections -->
<section data-dotscroll>Section 1</section>
<section data-dotscroll>Section 2</section>

<!-- 3. Include JS (auto-initializes) -->
<script src="dotscroll.js"></script>
03

Zero JS Setup

Configure via HTML.

Change options directly on the script tag using data attributes.

<!-- Disable smooth scroll -->
<script src="dotscroll.js"
  data-smooth="false"></script>

<!-- Position on the left -->
<script src="dotscroll.js"
  data-position="left"
  data-offset="32"></script>
04

JavaScript API

Full control.

Need more power? Initialize manually and use the full API.

// Manual init with options
const ds = DotScroll.init({
  smooth: true,
  ease: 0.06,
  onChange: (i, el) => {
    console.log('Section', i);
  }
});

// Control programmatically
ds.scrollTo(2);
ds.setSmooth(false);
