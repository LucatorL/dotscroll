Instalación

3 formas de empezar.

Elige la que mejor se adapte a tu proyecto. Todas funcionan igual.

La forma más fácil. Copia y pega en tu HTML:

HTML 
<!-- 1. Añade el CSS en el <head> -->
<link rel="stylesheet"
  href="https://unpkg.com/dotscroll@1.0.0/dist/dotscroll.min.css">

<!-- 2. Marca tus secciones -->
<section data-dotscroll>Hero</section>
<section data-dotscroll>About</section>
<section data-dotscroll>Contact</section>

<!-- 3. Añade el JS antes de </body> -->
<script src="https://unpkg.com/dotscroll@1.0.0/dist/dotscroll.min.js"></script>

<!-- ¡Listo! Funciona solo ✨ -->

Configurar sin JS

HTML Data Attributes.

Cambia el comportamiento sin escribir JavaScript. Perfecto para prototipos rápidos.

HTML 
<!-- Scroll en la izquierda, más fluido -->
<script src="dotscroll.min.js"
  data-position="left"
  data-ease="0.04"
  data-offset="32"></script>

<!-- Sin smooth scroll -->
<script src="dotscroll.min.js"
  data-smooth="false"></script>

<!-- Labels personalizados para accesibilidad -->
<section data-dotscroll data-dotscroll-label="Inicio">...</section>
<section data-dotscroll data-dotscroll-label="Nosotros">...</section>

JavaScript API

Control total.

Para cuando necesitas más poder. Inicializa manualmente y usa la API completa.

JavaScript 
// Desactiva auto-init en el script tag
// <script src="dotscroll.min.js" data-auto-init="false"></script>

// Inicialización manual
const ds = DotScroll.init({
  smooth: true,
  ease: 0.06,
  position: 'right',
  offset: 32,
  onChange: function(index, el, prev) {
    console.log('Sección activa:', index);
    // Cambiar color de fondo, analytics, etc.
  }
});

// Navegar a una sección
ds.scrollTo(2);

// Obtener la sección activa
const current = ds.getActiveIndex();
const element = ds.getActiveSection();

// Cambiar smooth scroll en tiempo real
ds.setSmooth(false);

// Recalcular después de cambios en el DOM
ds.refresh();

// Destruir la instancia
ds.destroy();

// Destruir todas las instancias
DotScroll.destroyAll();

Métodos de instancia

MétodoDescripción
scrollTo(index)Scroll suave a la sección indicada (0-based)
getActiveIndex()Devuelve el índice de la sección activa
getActiveSection()Devuelve el elemento DOM de la sección activa
setSmooth(bool)Activar/desactivar smooth scroll en tiempo real
refresh()Recalcular layout (llamar tras cambios dinámicos en el DOM)
destroy()Eliminar todo y limpiar eventos
CSS Variables

Personaliza todo.

Overridea las CSS variables en tu stylesheet para adaptar DotScroll a cualquier diseño.

  --dotscroll-bg: rgba(255,255,255,0.06);
  --dotscroll-dot-color: rgba(255,255,255,0.15);
  --dotscroll-fill-color: #ffffff;
  --dotscroll-counter-color: #fff;
}
